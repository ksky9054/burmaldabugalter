# Функция сброса пароля с кодовой фразой

## Описание

Добавлена функция сброса пароля с защитой через кодовую фразу "движуха". Это позволяет пользователю восстановить пароль, если он его забыл.

## Кодовая фраза

**Правильная фраза:** `движуха`

⚠️ **Важно:** Фраза чувствительна к регистру и должна быть введена точно

## Интерфейс

### Кнопка "Забыли пароль?" 
- Оранжевая кнопка на форме входа
- Откроет форму сброса пароля

### Форма сброса пароля включает:

1. **Поле "Логин"** - логин пользователя, у которого изменяется пароль
2. **Поле "Новый пароль"** - новый пароль (минимум 4 символа)
3. **Поле "Подтвердите пароль"** - подтверждение нового пароля
4. **Поле "Кодовая фраза"** - защита (слово "движуха")
5. **Счётчик попыток** - показывает оставшиеся попытки
6. **Кнопка "Сбросить пароль"** - выполнить операцию
7. **Кнопка "Отмена"** - закрыть форму

## Защита и ограничения

### Валидация
- ✅ Логин должен существовать в системе
- ✅ Новый пароль должен быть 4+ символа
- ✅ Пароли должны совпадать
- ✅ Кодовая фраза должна быть "движуха"

### Попытки входа
- **Максимум 3 попытки** введения неправильной кодовой фразы
- После 3-го неверного ввода форма закроется
- Счётчик показывает оставшиеся попытки (красный цвет при последней попытке)

### Сообщения об ошибках
- "Провьте кодовую фразу (подсказка: это слово, описывающее суету)"
- Нельзя узнать была ли фраза неверной или логин не найден

## Процесс сброса пароля

1. **На форме входа** нажмите кнопку "Забыли пароль?" (оранжевая)
2. **В форме сброса:**
   - Введите помню ваш логин
   - Введите новый пароль
   - Подтвердите пароль
   - **Введите кодовую фразу: "движуха"**
   - Нажмите "Сбросить пароль"
3. **Если всё верно:**
   - Сообщение "Пароль успешно сброшен!"
   - Вы вернётесь на форму входа
   - Теперь можете войти с новым паролем

4. **Если что-то неверно:**
   - Сообщение об ошибке
   - Счётчик попыток уменьшится
   - При 3-й ошибке форма закроется

## Пример использования

```

Результат: ✅ Пароль сброшен

```

```
Новый пароль: secure123
Подтвердить: secure123
Кодовая фраза: движуха
Результат: ❌ Ошибка (логин не найден), попыток осталось: 2
```

## Безопасность

### Что защищено
- 🔐 Пароль хешируется BCrypt перед сохранением
- 🔐 Требуется кодовая фраза
- 🔐 Максимум 3 попытки (защита от brute-force)
- 🔐 Нет информации о праве ошибку (логин или фраза)

### Рекомендации
- ✅ Используйте сложные пароли (буквы, цифры)
- ✅ Кодовая фраза известна только администратору
- ✅ Логируйте попытки сброса пароля (можно добавить)

## Изменение кодовой фразы

Чтобы изменить кодовую фразу:

1. Откройте **ResetPasswordForm.cs**
2. Найти строку:
```csharp
if (passphrase != "движуха")
	return false;
```
3. Измените "движуха" на нужную фразу
4. В DatabaseManager.cs также обновите:
```csharp
if (passphrase != "движуха")
	return false;
```
5. Пересоберите проект
