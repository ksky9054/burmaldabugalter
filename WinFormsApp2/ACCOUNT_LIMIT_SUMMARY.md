# ограничение на 2 аккаунта - СВОДКА

## ✅ Что реализовано:

### 1. **Ограничение на максимум 2 аккаунта**
- Константа `MAX_ACCOUNTS = 2` в DatabaseManager.cs
- Проверка перед созданием каждого аккаунта
- Автоматическое отклонение 3-го и последующих аккаунтов

### 2. **Методы управления лимитом в DatabaseManager**
```csharp
public int GetUserCount()              // Количество активных аккаунтов
public bool IsAccountLimitReached()    // Достигнут ли лимит?
public int GetRemainingSlots()         // Сколько слотов осталось?
public int GetMaxAccounts()            // Максимум аккаунтов
```

### 3. **Поведение формы регистрации**
- При загрузке проверяет лимит
- Если лимит достигнут (2/2) - форма закроется с сообщением
- В заголовке показывает оставшиеся слоты: "Регистрация (осталось слотов: 1)"

### 4. **Поведение формы входа**
- Кнопка обновляется при загрузке
- Если слоты заполнены: кнопка отключена с текстом "Регистрация (лимит достигнут)"
- Если слоты есть: кнопка показывает "Создать новый аккаунт (X/2)"

### 5. **Сообщения об ошибках**
- Если достигнут лимит: "Достигнут лимит создания аккаунтов! Максимально допустимо: 2 аккаунта."
- При попытке 3-го аккаунта: "Достигнут лимит создания аккаунтов! Максимально допустимо: 2 аккаунта."

### 6. **Класс SystemInfo**
- Отображает полную информацию о состоянии системы
- Показывает визуальную прогресс-полосу
- Методы:
  - `DisplaySystemInfo()` - открыть диалог с информацией

### 7. **Документация**
- DATABASE_README.md - полное описание системы с ограничениями
- ACCOUNT_LIMIT_EXAMPLES.txt - примеры использования
- Комментарии в коде с XMLDoc

## 📊 Сценарии использования:

### Сценарий 1: Создание 1 аккаунта (1/2)
1. Пользователь нажимает "Создать новый аккаунт"
2. Форма регистрации открывается
3. Заголовок: "Регистрация (осталось слотов: 1)"
4. Аккаунт создан успешно
5. На форме входа: "Создать новый аккаунт (1/2)"

### Сценарий 2: Создание 2-го аккаунта (2/2)
1. Пользователь нажимает "Создать новый аккаунт"
2. Форма регистрации открывается
3. Заголовок: "Регистрация (осталось слотов: 0)"
4. 2-й аккаунт создан успешно
5. На форме входа: кнопка отключена с текстом "Регистрация (лимит достигнут)"

### Сценарий 3: Попытка создания 3-го аккаунта (2/2)
1. Пользователь нажимает "Создать новый аккаунт"
2. Форма регистрации закроется с сообщением об ошибке
3. Сообщение: "Достигнут лимит создания аккаунтов! Максимально допустимо: 2 аккаунта."
4. Возврат на форму входа

## 🔧 Как изменить лимит:

1. Откройте файл `DatabaseManager.cs`
2. Найти строку: `private const int MAX_ACCOUNTS = 2;`
3. Измените число на нужное (например, 5 для 5 аккаунтов)
4. Пересоберите проект
