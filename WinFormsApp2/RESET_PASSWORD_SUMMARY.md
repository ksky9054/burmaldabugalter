# Функция сброса пароля - СВОДКА

## ✅ Что реализовано:

### 1. **Форма сброса пароля (ResetPasswordForm)**
- Поле для логина
- Поле для нового пароля
- Подтверждение пароля
- **Поле для кодовой фразы** 🔒
- Счётчик попыток
- Кнопка "Сбросить пароль"

### 2. **Защита через кодовую фразу**
- **Правильная фраза:** `движуха`
- Максимум **3 попытки** ввода
- Форма закроется после 3-й ошибки
- Счётчик показывает оставшиеся попытки

### 3. **Метод в DatabaseManager**
```csharp
public bool ResetPassword(string username, string newPassword, string passphrase)
{
	// Проверяет фразу "движуха"
	// Валидирует пароль
	// Обновляет пароль в БД
	// Хеширует через BCrypt
}
```

### 4. **Кнопка на форме входа**
- Оранжевая кнопка: **"Забыли пароль?"**
- Откроет форму сброса пароля
- После сброса вернётесь на форму входа

### 5. **Валидация**
- ✅ Проверка существования логина
- ✅ Проверка кодовой фразы "движуха"
- ✅ Пароль минимум 4 символа
- ✅ Совпадение паролей
- ✅ Хеширование через BCrypt

## 📊 Процесс сброса:

```
1. Нажмите "Забыли пароль?" на форме входа
					⬇️
2. Откроется форма сброса пароля
					⬇️
3. Введите:
   - Логин
   - Новый пароль
   - Подтверждение
   - Кодовую фразу: "движуха" 🔑
					⬇️
4. Нажмите "Сбросить пароль"
					⬇️
5. Если все правильно:
   ✅ "Пароль успешно сброшен!"
   Вернётесь на форму входа

6. Если ошибка:
   ❌ "Ошибка сброса пароля!"
   Будет показана подсказка
   -1 попытка
```

## 🔐 Примеры:

### Успешный сброс:
```
Логин: admin
Новый пароль: MyNewPass123
Подтверждение: MyNewPass123
Кодовая фраза: движуха
✅ Результат: Пароль сброшен
```

### Неверная фраза (1/3):
```
Кодовая фраза: движение ❌
❌ Ошибка! 
Осталось попыток: 2
```

### После 3 ошибок:
```
(3-я неверная попытка)
❌ "Превышено максимальное количество попыток!"
Форма закроется
```
